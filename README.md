# Taskhelper-Hub_R-Shiny
A multi-utility R Shiny web application for operational data checks. Features include Veeva ID comparison (new, old, expired), DIO link validation, NCT differentiation, and PDF compilation. Designed to streamline internal workflows through a single user-friendly interface.
